Core Algorithms

MAPF + Traffic Control

A deep integration of pathfinding and traffic management, offering three complementary algorithms for different scenarios.

CBS (Conflict-Based Search)

Ideal for small-scale, conflict-sensitive planning scenarios with optimal solutions.
Performance: ~5 seconds for 20 robots.

Improved LaCAM2

Suitable for medium-scale, relatively fixed business scenarios. Quality improves with computation time.
Performance: 300 robots in ~1 second.

CLrp (Proprietary)

Most flexible, ideal for large-scale, highly dynamic scenarios with incremental computing.
Performance: 300 robots in ~1.5 seconds.

Resource Allocation Algorithm

Flexible time-window based allocation using a best-match algorithm based on priority and time. Applied for charging, workstation, and storage location assignments.
Performance: Allocates 1000 tasks in ~1 second.
